First priority:
For the target Car-Pillar, we want a map covering 4x4 arcmin.
In order to produce a spatially homogeneous map,
please run the following and repeat this procedure until the
allocated time of 7.5 hours (total) is used up:

off = -90 + 45 * arange(5)
repeat(1)
for i in range(5):
for j in range(5):
source 'Car-Pillar'
offset(x=off[i], y=off[j], unit='arcsec', system='EQ')
